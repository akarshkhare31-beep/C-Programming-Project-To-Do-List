# C-Programming-Project-To-Do-List 

The To-Do List Management System is a C program that helps user to manage their daily tasks efficiently.

It allows users to:
Add new tasks to the task list
View all added tasks
Delete completed or unwanted tasks
Display messages for valid and invalid user inputs

It uses structures, arrays, pointers, and functions to store and manage task data.

This project also solves common problems users face, such as:
Forgetting daily tasks
Difficulty in organizing work
Managing multiple tasks manually
